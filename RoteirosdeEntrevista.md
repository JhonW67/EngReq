# ROTEIRO DE ENTREVISTA PARA ELICITAÇÃO DE REQUISITOS

## 1. INTRODUÇÃO**
   
    Este documento apresenta um roteiro de entrevista semiestruturada elaborado com o objetivo de apoiar o processo de elicitação de requisitos para um Sistema de Gestão Acadêmica (SGA). A entrevista é direcionada à persona "Carlos", professor universitário que leciona múltiplas disciplinas e enfrenta dificuldades no lançamento de notas.
    
## 2. OBJETIVO DA ENTREVISTA
    
     as principais dificuldades enfrentadas pelo professor no uso do sistema atual, especialmente no processo de lançamento de notas, bem como levantar necessidades, expectativas e sugestões para o desenvolvimento de um novo sistema mais eficiente.

## 3. IDENTIFICAÇÃO
    Entrevistado: Carlos (Professor)
    Data: 24/03/2026
    Local: Centro Universitário
    Duração prevista: 60 minutos

## 4. ROTEIRO DE PERGUNTAS
   
    *Perguntas de Contexto*
        
        Qual é a sua função e quais disciplinas você leciona atualmente? (Aberta)
    
        Há quanto tempo você utiliza o sistema acadêmico atual? (Fechada)
    
        Como você utiliza o sistema no seu dia a dia? (Aberta)
    
    *Processo Atual (AS-IS)*
        
        Descreva como você realiza o lançamento de notas atualmente. (Aberta)
        
        Com que frequência você precisa lançar notas no sistema? (Fechada)
    
        Quais ferramentas ou recursos você utiliza além do sistema (ex.: planilhas)? (Aberta)
    
        Quais são as principais dificuldades que você enfrenta ao lançar notas? (Aberta)
    
        O sistema atual atende suas necessidades no lançamento de notas? (Fechada)
    
    *Necessidades e Expectativas (TO-BE)*
    
        Se pudesse mudar algo no processo de lançamento de notas, o que seria? (Aberta)
    
        Quais funcionalidades seriam indispensáveis para facilitar seu trabalho? (Aberta)
    
        Você gostaria que o sistema automatizasse alguma parte do processo? Qual? (Aberta)
        
        É importante que o sistema permita acesso via celular ou tablet? (Fechada)
        
        Qual seria um tempo ideal para o sistema responder ao lançar notas? (Aberta)
    
    *Validação (Fechamento)*
        
        Existe algo importante sobre o sistema que não foi abordado? (Aberta)
    
        Você recomendaria outro professor para contribuir com essa análise? (Aberta)

## 5. CONSIDERAÇÕES FINAIS
    
    Este roteiro segue a técnica do funil, iniciando com perguntas gerais de contexto e evoluindo para questões mais específicas relacionadas ao processo atual e às necessidades do entrevistado. A combinação de perguntas abertas e fechadas permite coletar tanto dados qualitativos quanto quantitativos, contribuindo para uma elicitação de requisitos mais completa e eficaz.
